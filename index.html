Open Golf is a simple open source golf GPS app. Allows users to create their own golf course definitions or use Free Mode. Currently in Beta with a VERY limited online course database of courses I personally play regularly.

Open golf will show you your yardage from anywhere in the world.

Please contact me if you would like to add courses to the online database. Open Golf only needs a single GPS point for the centre of each green. Open Golf only needs a single GPS point for the centre of each green.
